2.在打开一个终端

有一个文件为ngrok-v3-stable-linux-amd64.tgz

命令解压

tar -xzf ngrok-v3-stable-linux-amd64.tgz

给权限 chmod +x ngrok

./ngrok authtoken <你的Token>              # 已保存可跳过
./ngrok http 8000                          # 启动
